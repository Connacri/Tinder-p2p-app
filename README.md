## ✨ Fonctionnalités

- 📱 **App Android native** (APK)
- 🌐 **PWA** installable sur tous les appareils
- 👤 **Profils** avec photo, pseudo, bio
- 🔍 **Découverte** de profils en temps réel
- 💬 **Chat P2P direct** (Gun.js)
- ⭐ **Favoris** persistants
- 🔒 **100% décentralisé** - Aucun serveur

---

## 🚀 Installation Android

1. Télécharge l'APK depuis les [Releases](https://github.com/Connacri/Tinder-p2p-app/releases/latest)
2. Ouvre le fichier `.apk`
3. Accepte "Sources inconnues" si demandé
4. Lance l'application !

## 🛠️ Build automatique

Chaque push sur `main` :
- ✅ Génère un nouvel APK
- ✅ Crée une Release GitHub
- ✅ Incrémente la version automatiquement
